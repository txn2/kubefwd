# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.23.x  | Yes       |
| < 1.23  | No        |

## Reporting a Vulnerability

Email security concerns to Craig Johnston <cj@imti.co>.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You can expect an initial response within 72 hours.
